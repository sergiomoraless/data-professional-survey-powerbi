# Data Professional Survey — Power BI Dashboard

Analysis of a survey of 630 data professionals, built as a
Power BI learning project.

The original dataset comes from Alex The Analyst's tutorial.
Page 1 replicates the guided dashboard. Page 2 is independent analysis.

## Questions that guided the independent analysis

- Which programming language dominates in each country?
- Which industries offer the highest average salaries for data professionals?
- Is there a relationship between perceived difficulty breaking
  into data and current salary satisfaction?
- Does average salary vary significantly by age group
  within each job title?

## Key Findings

- India has the highest proportion of Python users (~86%),
  while the United States shows the highest proportion of R (~20%).
- Healthcare leads all industries with the highest average salary (~$64k),
  followed by Finance (~$58k) and Real Estate (~$56k).
- The harder it was to break into data, the lower the salary
  satisfaction: those who reported "Very Difficult" average 2.57/10
  vs 5.19/10 for "Very Easy".
- In Data Scientist and Data Engineer roles, the 30-39 age group
  earns notably more than the 20-29 group, suggesting experience
  carries real weight in these positions.
